## Installation

#### Front-End:

FoodTrip required [React] V16.0.3 to run.
Runs on PORT 3000.

```sh
$ npm install
```
**or:**
```sh
$ yarn install
```

#### Back-End:

FoodTrip required [Node.js] V12.18.03 to run.
Runs on PORT 4000 (for changing you need to set in the back and the front).

```sh
$ npm install
```
**or:**
```sh
$ yarn install
```

## Activated 

#### Front-End:

```sh
$ npm start
```
**or:**
```sh
$ yarn start
```

#### Back-End:

* run in nodemon.

```sh
$ npm run dev
```
**or:**
```sh
$ yarn run dev
```

* **run without nodemon.**

```sh
$ npm run start
```
**or:**
```sh
$ yarn run start
```
